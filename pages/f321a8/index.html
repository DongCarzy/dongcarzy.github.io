# history

查看历史命令

  • 通过 export HISTTIMEFORMAT='%F %T' 命令, 可以临时显示命令执行的时间
  • 通过 !${id} 可以快速执行上次的命令, eg: !12

# xargs

将前面的输出作为后面的输入命令.

在当前文件夹下面查询包含 ERROR 字符串的位置

find . -type f -print | xargs grep "ERROR"
